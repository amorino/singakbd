## Changelog

### R1 (2019)

#### Features

- 3mm integrated plate
- Brass weight
- QMK/VIA compatible
- PCB designed by Wilba
- USB-C

#### Colors

- Velvet
- Pink
- Navy
- Black
- Space Grey

### R2 (2022)

#### Changes from previous versions

- Top Mounted 1.5mm plate
- Hotswap H08 designed by hineybush
- Screw positions changed
- Slight changes on the Ocelot cat engraving

#### Colors

- Dark Green
- Indigo (Purple)
- Polycarbonate
- Black w/ Copper weight
